Welcome to my Project 0, this website will be a mixture of personal portfolio and who I am as a person. Each menu will have four to six subpages except for the contact page where it only has two and each subpage will go in depth and cover a topic. Some subpages will have images, icons, lists, and plain text. Starting with the about page which consists of six subpages called learn more about me, interests, basketball, badminton, videogames, and fast food. To provide you some context, basketball, badminton, and videogames are my hobbies. Next is the resume page which also has six subpages called summary, education, experience, web developer, psychology, and computer programming. Web developer is my major, I want psychology to be my minor, and computer programming as a career. Then we have the travel page which has 4 subpages named Philippines, Universal Studios, San Francisco, and Yosemite. The travel page contains multiple images. The penultimate is a portfolio page which is also my future projects that I want to work on, and it also consists of images. Lastly, we have the contacts page with two subpages called social profiles and email me. I got this template from BootstrapMade.
